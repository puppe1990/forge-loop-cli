# Code of Conduct

## Our Pledge

We are committed to a respectful, harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Focusing on technical discussion, not personal attacks
- Accepting feedback gracefully

Examples of unacceptable behavior:

- Harassment, insults, or discrimination
- Publishing private information without permission
- Bad-faith disruption of discussions

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and
other contributions that violate this Code of Conduct.

## Reporting

Report issues to the maintainer via GitHub issues using the `question` template.
